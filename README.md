## 🚀 Rodando localmente

Basta abrir o `index.html` no navegador:

```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
```

Ou rode um servidor local simples:

```bash
python3 -m http.server 8000
# acesse http://localhost:8000
```

## 🌐 Publicando no GitHub Pages

1. Crie um repositório (ex.: `portfolio`) e envie estes arquivos para a branch `main`.
2. No GitHub, vá em **Settings → Pages**.
3. Em **Source**, selecione `Deploy from a branch` → branch `main` → pasta `/ (root)`.
4. Salve. Em alguns minutos o site estará no ar em `https://seu-usuario.github.io/portfolio`.

## ✏️ Personalizando

- **Conteúdo:** edite os textos diretamente no `index.html`.
- **Cores e tipografia:** ajuste as variáveis no topo do `styles.css` (`:root`).
- **Links de contato:** atualize os `href` de email, LinkedIn e GitHub.
